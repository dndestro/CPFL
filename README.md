# Propósito

Este script realiza login no site da CPFL piratininga e recupera o consumo do último mês.

Funcionalidade ainda não implementada - o script adicionará o consumo dentro do Home Assistant


O username e a senha devem ser preenchidos no script.

A primeira vez que o script for executado, um arquivo com os cookis chamado auth.json é criado. Este procedimento facilita logins subsequentes.
